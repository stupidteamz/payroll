# ระบบจัดการเงินเดือน - ดาวรุ่ง ทราเวล

แอปพลิเคชันสำหรับบริหารจัดการข้อมูลพนักงาน, ยานพาหนะ, เส้นทางเดินรถ, การจัดตารางเวร และคำนวณสลิปเงินเดือนอัตโนมัติ

## 1. การเริ่มต้นใช้งาน

### ข้อกำหนดเบื้องต้น
- ติดตั้ง Node.js (เวอร์ชันล่าสุด)

### การติดตั้ง
1. เปิด Terminal ไปที่โฟลเดอร์ `D:\gemini-bot-payroll-system`
2. ติดตั้ง Dependencies ของ Backend:
   ```bash
   cd backend
   npm install
   cd ..
   ```
3. ติดตั้ง Dependencies ของ Frontend:
   ```bash
   cd frontend
   npm install
   cd ..
   ```

## 2. วิธีการเริ่มระบบ (Run)

เพื่อให้ระบบทำงานเต็มประสิทธิภาพ แนะนำให้รันแยกหน้าต่าง:

**หน้าต่างที่ 1 (Backend):**
```bash
cd backend
node server.js
```

**หน้าต่างที่ 2 (Frontend):**
```bash
cd frontend
npm run dev
```
จากนั้นเปิดเบราว์เซอร์ไปที่ลิงก์ที่แสดงในหน้าต่าง Frontend (เช่น `http://localhost:5173/`)

---

## 3. เมนูการใช้งาน

### 3.1 จัดการข้อมูลหลัก (Master Data)
- **พนักงาน:** เพิ่ม, ค้นหา, ลบข้อมูลพนักงาน และตั้งค่าเงินเดือน/อัตรา OT
- **ยานพาหนะ:** เพิ่มข้อมูลรถ, สถานะการใช้งาน
- **เส้นทาง:** เพิ่มเส้นทางเดินรถ

### 3.2 ตารางการทำงาน (Schedule Matrix)
- เลือกเส้นทางและยานพาหนะ
- ลงตารางเวรพนักงาน (เช้า, บ่าย, OT1, OT2) โดยคลิกเลือกช่องที่ต้องการ

### 3.3 สลิปเงินเดือนและรายงาน
- **สร้างสลิปเงินเดือน:** เลือกเดือน/ปี แล้วกด "Generate" ระบบจะคำนวณอัตโนมัติ
- **รายงานการเดินรถ:** ดูสรุปจำนวนเที่ยวเดินรถ แยกตามเส้นทางและยานพาหนะ
- **รายงานการจ่ายเงิน:** ดูภาพรวมค่าใช้จ่ายเงินเดือนทั้งหมดของบริษัท

---
*หากพบปัญหาการใช้งาน สามารถตรวจสอบ log ได้ในหน้าต่าง Terminal ของ Backend*
